# TAREA-PORT-SCANNER-BANNER-GRABBER
Este repositorio servira para escanear puertos para saber si estan abiertos y adicional capturar informacion del servicio que corre cada uno de estos puertos abiertos.
Esto es solo una evaluacion.
# HOW TO USE
Lo primero que deben de tener es Kali linux instalado mediante virtual box o alguna maquina virtual.
Luego instalar python abriendo el terminal de kali linux y usando el comando sudo apt install python3.
Clona este repositorio, modifica el script "port-scanner-banner-grabber" en caso de ser necesario
# Nota: Si sera necesario porque necesitaras cambiar a la direccion Ip requerida
Mediante el terminal navega hasta la carpeta donde descargaste el script y ejecutalo con el comando python3 al inicio
# Ejemplo: python3 port-scanner-banner-grabber.py
